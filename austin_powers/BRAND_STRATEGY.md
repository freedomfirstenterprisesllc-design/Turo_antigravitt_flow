## ðŸ…°ï¸ LANE A â€” SERIOUS RELEASES (Detailed)

### Definition

Playlist-ready singles with:

- **Strong hooks** â€” memorable, singable
- **Clean mixes** â€” professional, radio-ready
- **Emotional clarity** â€” listener feels something

### Release Specifications

| Element | Requirement |
|---------|-------------|
| Frequency | 1 single every 3-4 weeks |
| Length | Standard (2:30-4:00 typical) |
| Mix Quality | Professional (may need external mix/master) |
| Artwork | High-quality, cohesive with brand |
| Metadata | Complete (ISRC, credits, lyrics) |
| Lead Time | 2-3 weeks before release for distribution |

### Distribution Checklist (Per Serious Release)

- [ ] Final master approved
- [ ] Album artwork (3000x3000, 300dpi)
- [ ] Metadata complete (title, artist, genre, ISRC)
- [ ] Lyrics submitted
- [ ] Release date set (2+ weeks out)
- [ ] Pre-save campaign active
- [ ] Playlist pitching submitted (3+ weeks before release)
- [ ] Social announcement scheduled
- [ ] Music video / visual planned

### Success Metrics for Lane A

- Spotify algorithmic playlist adds
- Editorial playlist submissions
- Save rate (listeners who save the track)
- Repeat listen rate
- Monthly listener growth

## ðŸ…±ï¸ LANE B â€” MALODY RELEASES (Detailed)

### Definition

Slightly comical or clever "malodies" â€” parody-adjacent content built from existing ideas/assets when possible.

### Purpose

- **Reach new listeners fast** (virality potential)
- **Drive traffic to serious releases** (the funnel)
- **Low production overhead** (repurpose existing material)
- **High volume output** (content machine)

### Format Rules (NON-NEGOTIABLE)

| Rule | Specification | Why |
|------|---------------|-----|
| Length | 30-120 seconds | Short-form algorithm optimization |
| Hook Start | Within 3-5 seconds | Immediate engagement, prevent scroll |
| Punchline | By 15-25 seconds | Payoff before viewer drops off |
| Audio | Professional quality | Credibility, even in comedy |

### Cadence Options

- **Option A:** 1 malody drop per week
- **Option B:** 2 malody clips per week

### Source Priority (Use What We Have)

1. Extract hooks from existing catalog
2. Rebuild structure (shorter intro, faster payoff)
3. Repurpose existing video into 10-30 sec clips
4. Convert song concepts into malody format
